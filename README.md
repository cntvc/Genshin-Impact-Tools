# 原神小工具


## 主要功能点：
- [X] 抽卡记录导出（由于官方限制，只能导出最近6个月抽卡记录）    
  - 可选的多种导出方式：剪切板、游戏缓存、配置文件、抓包（未完成）
- [ ] 历史记录合并(1.3版本前产生的记录尚未支持合并)
- [ ] 米游社签到
- [ ] 待开发

其他特点：
- [X] 多账户支持
- [ ] 国际服支持

软件支持的系统版本：win7，win10， win11
